# 运动方案模块说明

## 功能点

- 采集用户健康档案（年龄、身高、体重、慢性病史、目标）
- 调用 `/api/ai/exercise/plan` 接口生成个性化方案
- 前端解析 JSON，展示周计划与注意事项

## Prompt 策略

- 在 `ExercisePlanController` 中构造结构化提示词（JSON 模板）
- 按照用户可用训练天数生成对应的周计划节点
- 支持后续扩展（如强度级别、心率区间等）

## 前端处理

- 表单数据验证后发起请求
- 优先解析模型返回的 JSON；若解析失败则使用原始文本展示
- 预留提示信息供用户参考

## 后续规划

1. 将方案落库至 `exercise_plan`、`exercise_plan_schedule` 表。
2. 增加方案打卡功能，对应 `exercise_record` 表。
3. 引入数据可视化（体重曲线、运动时长统计）。
4. 支持方案调整与医生评审流程。
