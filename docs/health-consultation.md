# 健康咨询模块设计

## 功能概述

- 基于 `langchain4j` + 通义千问的智能问答
- 支持多轮对话、历史记录、收藏
- 可扩展医疗知识库进行 RAG 检索

## 现状

- 后端提供 `POST /api/ai/consult/chat` 接口（见 `MedicalConsultController`）。
- 前端页面 `ConsultView` 已对接接口，支持基础问答。
- 数据库表 `health_consultation`、`consultation_message` 已建库脚本。

## 后续落地

1. **会话管理**：
   - 创建会话记录，保存用户问题与 AI 回复。
   - 支持分页查询、导出。

2. **知识库准备**：
   - 引入权威医疗文档，向量化后存入 PGVector。
   - 定期更新基础知识。

3. **安全控制**：
   - 输出免责声明、风险提示。
   - 内容审核（敏感词、违规用语）。

4. **专家介入**：
   - 支持人工医生接管高风险会话。
   - 通知后台审核模块。
